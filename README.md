## 📁 Repository Structure

Solutions are organized by contest and problem, following this pattern:

```
├── Contest_XXXX/
│   ├── A.cpp
│   ├── B.cpp
│   ├── C.py
│   └── ...
├── Contest_YYYY/
│   ├── A.cpp
│   └── ...
└── README.md
```

Each folder corresponds to a contest (named by its Codeforces contest number), and each file corresponds to a problem (named by its problem letter/index).
